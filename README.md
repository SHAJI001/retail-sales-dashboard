# Retail Sales Performance Dashboard

## Overview
This project analyzes retail transaction data to understand sales, profit, and trends across regions and product categories. It includes an interactive Excel/Google Sheets dashboard that summarizes key KPIs and provides clear insights for business stakeholders.

## Repository Structure

## Tools Used
- Excel / Google Sheets  
- Pivot Tables  
- Charts (bar, line, KPI cards)  

## Key Questions Answered
- Which regions generate the most sales and profit?  
- Which categories are most profitable?  
- What are the monthly sales trends?  
- Are any segments underperforming?  

## How to Use
1. Download the dataset from the **/data** folder.  
2. Open it in Excel or Google Sheets.  
3. Build the pivot tables listed in the case study.  
4. Create visuals and organize them into a dashboard.  
5. Add slicers to make it interactive.  

For detailed analysis steps, see **CASE_STUDY.md**.
